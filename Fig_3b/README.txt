To reproduce the Charts in Fig. 3(b):

- go to the CODE folder

- run the simulations for the periodic geometries
	> for the geometry files, please contact giacomo.falcucci@uniroma2.it
          (the files are 1 Gb each)
	> load the input files in the CODE/RUN folder:
		bgk.A.input	--> Cylinder
		bgk.B.input	--> Cylinder with Ridges
		bgk.C.input	--> Reticular Sponge
		bgk.D.input	--> Euplectella Aspergillum (periodic)

- open Paraview (version 5.6.0 or more recent)

- load Fig_3b.pvsm state file
